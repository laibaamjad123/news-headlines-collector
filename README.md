# 📰 News Headlines Collector

A Python script that fetches real-time top news headlines 
using NewsAPI and saves them to a clean CSV file.

## 🛠️ Tools Used
- Python
- Requests
- NewsAPI (newsapi.org)

## 📊 Output
Clean CSV file with:
- Article title
- Source name
- Published date & time
- Article URL

## ▶️ How to Run
pip install requests
python news_scraper.py

## 🔑 API Key
Get your free API key from newsapi.org
Replace "YOUR_API_KEY" in the code with your actual key
